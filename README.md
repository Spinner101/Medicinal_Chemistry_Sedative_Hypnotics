# Medicinal Chemistry of Sedative-Hypnotics

Public-facing teaching repository for PY1 PharmD students.

## Medicinal Chemistry Framework
Lectures emphasize:
- Pharmacophore recognition
- Physicochemical properties
- Structure–activity relationships (SAR)
- Drug–receptor interactions
- Metabolism applied to therapeutic decision-making

## Therapeutic Area
Sedative–Hypnotics

## Lecture Modules
1. Benzodiazepine Sedative-Hypnotics
2. Non-Benzodiazepine Sedative-Hypnotics (Z-drugs)
3. Alternative Hypnotic Pathways (Melatonin, Orexin, Barbiturates)

## Rendering
Lectures are written in Markdown and compatible with Marp for slide rendering.
