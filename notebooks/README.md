# Notebooks (Instructor-Run)
These notebooks generate dynamic medicinal chemistry assets (PubChem images, drugâ€“target maps, SAR/metabolism tables).

## Student access model
Students are **not expected to run notebooks**. Instructor/TA runs notebooks and posts:
- rendered HTML/PDF notebook
- slide-ready figures (`figures/slide_ready/`)
- CSV tables (`data/`)

## Run locally
1. Install Python 3.10+
2. Install dependencies:
   - `pip install -r requirements.txt`
3. Open Jupyter:
   - `jupyter lab` or `jupyter notebook`
4. Run all cells

## Outputs
- `figures/` and `figures/slide_ready/`
- `data/` (CSV + manifest JSON)
